# 02582 Computational Data Analysis - Case 2

This repository contains the Case 2 analysis materials for the EmoPairCompete wearable biosignal dataset.

## Contents

- `submission/case2_analysis.py`: main reproducible analysis script.
- `submission/case2_extended_analysis.py`: additional analyses used for poster material.
- `submission/case2_report.tex`: report source.
- `submission/case2_report.pdf`: generated report.
- `submission/figures/`: figures used in the report.
- `submission/tables/`: generated numerical summaries.
- `submission/*poster*`: poster PDFs/PNGs and scripts used for the presentation.

## Data

The raw dataset is not included in this repository because the course data are governed by licensing restrictions.
To reproduce the analysis, place the provided `HR_data_2.zip` file in the repository root.

## Reproduce

Run from the repository root:

```powershell
python .\submission\case2_analysis.py
```

To rebuild the report PDF generated by the local ReportLab script:

```powershell
python .\submission\build_report_pdf.py
```

## Notes

The report includes an acknowledgement of GenAI use. Generated videos and raw data are intentionally excluded from version control.
